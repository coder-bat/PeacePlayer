## How to Use This System

### For Claude:
1. **ALWAYS read from and write to the Obsidian vault** (`.obsidian-memory/`)
2. **NEVER use the old memory file structure** (individual .md files in memory/)
3. Use **Obsidian-style links**: `[[Note Name]]` or `[[Note Name|Display Text]]`
4. Apply **tags** for categorization: `#user`, `#feedback`, `#project`, `#reference`
5. Use **frontmatter** for metadata in all notes

### For User:
- Open `.obsidian-memory/` as an Obsidian vault
- Use templates from `07_TEMPLATES/` for new notes
- Process inbox items regularly (move from 01_INBOX to appropriate folder)
- Link related notes using `[[WikiLinks]]`
